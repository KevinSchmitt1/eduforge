# Learner profile — beginner web developer

Example of a beginner-oriented profile. Pass it with
`eduforge build --topic "..." --profile examples/profiles/web-dev-beginner.md`.

## Who the learner is
A self-taught web developer comfortable with HTML/CSS/JavaScript and basic Python,
newer to computer-science fundamentals. Wants concrete, visual, runnable examples and
plain-language explanations without heavy mathematical notation.

## Already knows well — do NOT re-explain these
- Basic programming: variables, loops, functions, arrays/lists, dictionaries
- Reading and running Python in a notebook
- Installing packages with pip

## Still building up — DO explain these carefully
- CS fundamentals: data structures, algorithmic complexity (Big-O), recursion
- Any mathematical notation — prefer words and examples over formulas
- The reasoning behind *why* an approach is efficient or correct

## Environment / prerequisites the material must target
- A standard laptop, CPU only. Python 3.10+ in Jupyter.
- Standard library only where possible; avoid heavy dependencies.
- Keep examples small and visual; print intermediate steps so the learner can follow.
